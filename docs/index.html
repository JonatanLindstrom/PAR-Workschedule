PARPAS - Kalenderkonverterare

Detta program konverterar en av PARPAS genererad PDF till iCalendar-fil (ICS). En ICS-fil kan importeras i de flesta sorters kalenderprogram, exempelvis Google Calendar för att lättare hålla koll på ditt schema då krångel med PARPAS uppstår.

Notera att det lämnas ingen garanti på att programmet fungerar felfritt. Då jag endast haft mitt egna schema att testa programmet med kan oväntade problem uppstå för scheman med andra sorters pass eller från andra parker. Var därför noga med att dubbelkolla att ICS-filen stämmer överens med PDFen. Vid uppdateringar i PARPAS måste din kalender uppdateras manuellt.

Hämta program (.exe) Hämta source code (.py)


När du kör programmet kommer du få upp ett terminal-fönster samt en fil-dialog. Välj här det schema du fått från parpas eller i ditt anställningserbjudande. Programmet skriver nu om denna info till en iCalendar-fil och lägger i samma mapp som pdf-filen. Denna kan du nu importera i valfritt kalender program.
